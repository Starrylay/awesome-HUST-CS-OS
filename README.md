# 华中科技大学OS课程设计
华中科技大学19级沈承磊的OS课设/操作系统课程设计/Linux任务监视器/新增系统调用
## 开发环境：
### 1. 硬件环境：
a) 中央处理器 CPU：AMD Ryzen 7 5800H with Radeon Graphics 3.20 GHz
b) 物理内存 RAM：16GB
### 2. 编译环境：
a) 虚拟机软件：VMware Workstation 16 Pro
b) 虚拟机系统版本：Ubuntu14.04 操作系统
c) 内核：Linux4.4.10
d) 虚拟机核数：8
e) 虚拟机内存：4.00 GB
f) 虚拟机磁盘：60 GB
g) 编译器版本：GCC 7.5.0
h) 调试器版本：GNU gdb 8.1.0
